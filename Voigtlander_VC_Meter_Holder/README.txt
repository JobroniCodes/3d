# Summary

This lets you mount the VC meter on your strap incase you need the hot-shoe for flash. The meter can be installed/removed while the holder is on the strap, to do so simply slide half way and slide the vc meter in or out. 
Update: New clip has been added for the Peak Design Leash strap. 

# Print Settings

Printer: Malyan M180
Rafts: No
Supports: Yes
Resolution: 100 Micron 
Infill: 15%

Notes: 
Printed this with Hobbyking ABS and the meter should fit snuggly. Have not tried it with PLA, you can scale it if need be.
